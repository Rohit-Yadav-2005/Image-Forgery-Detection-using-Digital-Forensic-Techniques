📷 Digital Image Tampering Detection using Forensic Analysis

A Digital Image Processing (DIP) project implemented in MATLAB that detects potential tampering in images using classical image forensics techniques such as Error Level Analysis (ELA), Gradient Flow Analysis, and Noise Extraction.

The system analyzes inconsistencies introduced during image manipulation and highlights suspicious regions while providing a confidence score and tampering verdict.

📌 Project Overview

Digital images can be easily manipulated using modern editing tools. Detecting such modifications is important in fields such as:

Digital Forensics
Journalism & Media Verification
Legal Evidence Validation
Security & Surveillance

This project implements a forensic-based detection pipeline that identifies possible tampered regions in an image using multiple analysis techniques.

Instead of deep learning models, this project uses classical image processing methods, making it:

Lightweight
Interpretable
Computationally efficient

🔬 Forensic Techniques Used
1️⃣ Error Level Analysis (ELA)

ELA identifies inconsistencies in JPEG compression.

When an image is edited and saved again, different regions may have different compression levels. By recompressing the image and comparing it to the original, suspicious areas become visible.

Key Idea

ELA = |Original Image − Recompressed Image|

Suspicious regions appear brighter in the ELA map.

2️⃣ Gradient Flow Analysis

Gradient analysis detects unnatural edges and structural inconsistencies.

Tampered objects often introduce edges that do not match the surrounding image structure.

Gradient magnitude highlights such boundaries.

G = √(Gx² + Gy²)

This allows the system to detect unnatural transitions introduced during editing.

3️⃣ Noise Extraction

Digital cameras introduce natural sensor noise patterns.

When an object is inserted or edited:

The noise pattern becomes inconsistent
Edited regions often contain different noise characteristics

A high-pass filter is used to extract high-frequency noise components.

🧠 Detection Mask Generation

The outputs from:

ELA
Gradient Analysis
Noise Extraction

are combined into a single anomaly score map.

A statistical threshold is applied:

Threshold = Mean + 2 × Standard Deviation

Pixels exceeding the threshold are marked as suspicious.

The resulting binary image is called the Detection Mask.

🧹 Mask Refinement

To improve detection quality, the following image processing operations are applied:

Small region removal
Morphological closing
Hole filling
Region property filtering

This removes noise and retains only meaningful tampered regions.
